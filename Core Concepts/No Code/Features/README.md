-----Java features-----
Core Java Features:
1-Object-Oriented Programming: Java organizes code into objects for better structure and management.
2-Platform Independence: Java code runs on any system with a Java Virtual Machine (JVM), enabling cross-platform compatibility.

Performance and Safety:
3-Strong Typing: Java enforces strict type checks during compilation to reduce errors.
4-Automatic Memory Management: Java handles memory allocation and garbage collection automatically.
5-Multithreading: Java supports concurrent execution of multiple threads for efficient performance.

Development Ecosystem:
6-Comprehensive Standard Library: Java provides built-in tools and libraries for common programming tasks.
7-Built-in Security Features: Java includes mechanisms to protect applications from security risks.
8-Development Tools and Frameworks: Java offers IDEs, build tools, testing frameworks, and application servers to facilitate development and deployment.
9-Java for microservices: Microservices architecture breaks down a large application into smaller, independent services, each responsible 
for a specific business function. These services communicate with each other over a network, often using lightweight protocols like HTTP/REST.

These features make Java versatile for web, mobile, enterprise, cloud-native, and microservices-based applications.
