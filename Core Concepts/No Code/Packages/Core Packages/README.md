# Essential Java Packages Cheat Sheet
A clean, high-density reference guide mapping out the foundational packages built into the Java standard library. 
---

## 📦 Core Packages at a Glance
### 1. `java.lang` — The Foundation
* **Purpose**: Absolute core classes required to run any Java program.
* **Key Feature**: Automatically imported by the compiler. No explicit import statement needed.
* **Core Components**: `String`, `Math`, `System`, `Object`, `Exception`.

### 2. `java.util` — Utilities & Collections
* **Purpose**: Data structures, framework architectures, and legacy date utilities.
* **Core Components**: 
  * *Lists & Maps*: `ArrayList`, `HashMap`, `LinkedList`
  * *Utilities*: `Collections`, `Arrays`, `Optional`
  * *Legacy Date*: `Date`, `Calendar`

### 3. `java.io` — Input & Output
* **Purpose**: Data streams, serialization, and local file system interactions.
* **Core Components**: `File`, `InputStream`, `OutputStream`, `BufferedReader`, `BufferedWriter`.

### 4. `java.time` — Modern Date & Time
* **Purpose**: Modern, immutable, and thread-safe API for calendar calculations.
* **Core Components**: `LocalDate`, `LocalTime`, `LocalDateTime`, `ZonedDateTime`, `Duration`.

### 5. `java.net` — Networking
* **Purpose**: Low-level and high-level networking abstractions for web communication.
* **Core Components**: `Socket`, `ServerSocket`, `URL`, `URI`, `HttpURLConnection`.

### 6. `java.sql` — Database Connectivity (JDBC)
* **Purpose**: Standard API to connect Java applications to relational databases.
* **Core Components**: `Connection`, `Statement`, `PreparedStatement`, `ResultSet`, `DriverManager`.
