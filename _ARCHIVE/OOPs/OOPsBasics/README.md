# Java OOP Basics: Student Demo

This project shows how basic Object-Oriented Programming (OOP) works in Java. It highlights how to create blueprint classes and use different constructors.

## 🔑 Key Concepts Covered

* **Classes and Objects**: `Student` is the blueprint (class). `s1`, `s2`, and `s3` are the real instances (objects) built from it.
* **Constructor Overloading**: The `Student` class has three different constructors. This lets you create a student object in three different ways depending on what information you have.
* **The `this` Keyword**: Used to tell Java that we are talking about the variable belonging to *this specific object*, not the temporary variable in the method input.
* **Constructor Chaining**: The empty `Student()` constructor uses `this(...)` to call the first constructor. This saves you from rewriting code!

## 🚀 How to Run

1. Open a tool like JDoodle or your own computer terminal.
2. Copy the code from `MyClass.java` into your workspace.
3. Run the code to see the printout for all three different students.
