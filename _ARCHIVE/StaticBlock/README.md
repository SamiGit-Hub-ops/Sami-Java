### Description
This project demonstrates how **Static Blocks** and **Static Variables** work in Java. It highlights how memory is shared across different instances of a class.

### Key Concepts Covered
* **Static Block Execution:** Code that runs exactly once when the class is loaded into memory.
* **Shared Class State:** Showcases how modifying a static variable through one object instance changes the value globally for all other instances.
* **Direct Access:** Demonstrates accessing class-level static fields without initializing an object.
