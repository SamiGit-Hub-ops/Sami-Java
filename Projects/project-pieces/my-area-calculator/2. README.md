# Area & Volume Calculator

A lightweight Java command-line interface application designed to calculate the areas of 2D shapes and evaluate the volumes of 3D geometric structures.

## Features

- **2D Area Calculations:** Fully accessible through the user interface menu loops.
  - Circle
  - Square
  - Rectangle
- **3D Volume Core Logic:** Pre-compiled methods structured for backend evaluation.
  - Sphere
  - Cube
  - Cuboid

## Core Java Concepts Implemented

- **Object-Oriented Structure & Encapsulation:** Unifying functional state (`pi`) and modular shape configurations under a strict class boundary using `private` visibility tags.
- **Static Memory Allocation & Lifecycle:** Leveraging `static` properties allowing direct class executions completely detached from object instantiation phases.
- **Primitive Data Layouts & Floating-Point Suffixes:** Deploying single-precision storage types (`float`) flagged explicitly using literals (`f`) to maintain balanced platform overhead profiles.
- **Control Flow Frameworks:** Running persistent operations through nested logical conditional boundaries (`if-else if-else`) placed inside an un-broken evaluation chain (`while(true)`).
- **Console I/O Stream Processing:** Collecting live user interactions by linking structural platform utilities (`java.util.Scanner`) directly to the console input pipeline (`System.in`).
- **Data Architecture Transformations & Wrapper Classes:** Translating character inputs into numbers using static string translation tools (`Integer.parseInt`, `Float.parseFloat`).
- **Robust Multi-Catch Exception Structures:** Handling runtime anomalies gracefully via a multi-catch syntax block to catch `NumberFormatException`, `NoSuchElementException`, and `IllegalStateException`.
- **Explicit Numerical Operations Rules:** Eliminating logical processing truncation bugs within standard integer divisions by explicitly enforcing decimal definitions (`4.0f / 3.0f`).
- **Centralized System Memory Safety:** Consolidated resource lifecycle logic by placing `.close()` entirely outside the control loop execution flows, cleanly handling every possible application exit door with zero code repetition.

## Prerequisites

- **Java Development Kit (JDK):** Version 8 or higher.

## Getting Started

### Compilation
Compile the main source code file using the Java Compiler:
```bash
javac AreaCalculator.java
```

### Execution
Run the compiled application:
```bash
java AreaCalculator
```

## Internal Architecture Notes
- **Floating-Point Precision:** The application explicitly uses `float` primitives (`3.142f`) to handle structural metrics with balanced memory consumption.
- **Integer Division Correction:** The sphere calculation uses explicit float notation `(4.0f / 3.0f)` to prevent the Java compiler from executing truncation into an integer value of `1`.
- **UI Boundary:** The 3D calculations (`cube`, `cuboid`, `sphere`) are maintained strictly within the code scope and intentionally omitted from the interactive loop console menu mapping.
